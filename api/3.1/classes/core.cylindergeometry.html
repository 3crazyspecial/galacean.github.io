  • Overrides BufferGeometry.constructor

    Parameters

    • Optional radiusTop: number

      顶部圆柱的半径。 默认值为1。

    • Optional radiusBottom: number

      底部圆柱的半径。 默认值为1。

    • Optional height: number
    • Optional radialSegments: number

      圆柱体圆周周围的分割面数。 默认值为8

    • Optional heightSegments: number

      沿圆柱高度的面的行数。 默认值为1。

    • Optional openEnded: boolean

      一个布尔值，指示圆柱的末端是打开还是加盖。 默认值为false，表示上限。

    • Optional thetaStart: number

      第一段的起始角度，默认= 0（三点钟位置）。

    • Optional thetaLength: number

      圆形扇区的中心角，通常称为theta。 默认值为2 * Pi，这样可以获得完整的柱面。

    • Optional engine: Engine
