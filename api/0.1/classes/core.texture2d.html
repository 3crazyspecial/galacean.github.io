setPixelBuffer

  • setPixelBuffer(colorBuffer: ArrayBufferView, mipLevel?: number, x?: number, y?: number, width?: number, height?: number): void
    • Defined in packages/core/src/texture/Texture2D.ts:83

    通过颜色缓冲数据、指定区域和纹理层级设置像素，同样适用于压缩格式。 如果为WebGL1.0平台且纹理格式为压缩格式，第一次上传必须填满纹理。

    Parameters

    • colorBuffer: ArrayBufferView
    • Default value mipLevel: number = 0

      纹理层级

    • Optional x: number

      数据起始X坐标

    • Optional y: number

      数据起始Y坐标

    • Optional width: number

      数据宽度。如果为空的话 width 为 mipLevel 对应的宽度减去 x , mipLevel 对应的宽度为 Math.max(1, this.width >> mipLevel)

    • Optional height: number

      数据高度。如果为空的话 height 为 mipLevel 对应的高度减去 y , mipLevel 对应的高度为 Math.max(1, this.height >> mipLevel)

    Returns void
