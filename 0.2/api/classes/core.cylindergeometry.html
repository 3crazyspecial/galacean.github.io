    • Defined in packages/core/src/geometry-shape/CylinderGeometry.ts:16

    Parameters

    • engine: Engine

      Engine

    • Default value radiusTop: number = 1

      Cylinder top radius, default is 1.

    • Default value radiusBottom: number = 1

      Cylinder bottom radius, default is 1.

    • Default value height: number = 1

      Cylinder height, default is 1.

    • Default value radialSegments: number = 8

      Number of segmented faces around the circumference of the cylinder. Default is 8

    • Default value heightSegments: number = 1

      Number of rows of faces along the height of the cylinder. Default is 1.

    • Default value openEnded: boolean = false

      A boolean indicating whether the ends of the cylinder are open or capped. Default is false, meaning capped.

    • Default value thetaStart: number = 0

      Start angle for first segment, default = 0 (three o'clock position).

    • Default value thetaLength: number = 2 * Math.PI

      The central angle, often called theta, of the circular sector. The default is 2*Pi, which makes for a complete cylinder.

    • Default value frontFace: FrontFace = FrontFace.CCW
